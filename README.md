# 🔐 Secure Encryption Tool

A modern encryption tool built with FastAPI and a glassmorphism UI.

## 🚀 Features
- AES-256 encryption
- Text encryption & decryption
- File encryption (all file types)
- Drag & drop UI
- Progress bar support

## 🛠 Tech Stack
- Python (FastAPI)
- HTML/CSS/JS
- Cryptography (AES-GCM)
